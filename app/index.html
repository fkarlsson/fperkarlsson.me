Website: AgetoMTT (2012)
HTML / CSS / JS / PHP / Design

Working in collaboration with a friend of mine, she provided hand-drawn illustrations and feedback of my design decisions while I worked on the rest.

The design was mocked up in Photoshop and then realized with H5BP/960gs using MODx as back-end provider. The customer now maintan the site themselves through the MODx admin interface.

I chose MODx because of its flexibility and because I was already familiar with it. I will probably not use that CMS again in the future though as there are many better options available today.

iOS App: ASCC (2013)
Obj-C / Cocoa Touch / PHP (API)

My first approved iOS app which is used for viewing the customers data from a Weighing In Motion/Wheel Flat Detection System for the railway.

The app features some fancy table views, a sidebar menu, REST API authentication/access (OAuth) and some CoreGraphics drawing.

Using CoreData the app also stores all recent data on the memory, enabling offline support.

Note: Not really compatible with iOS 7 (as I am no longer employed to update this app)

Check it out!
Website: Popkalendern (2012)
HTML / SCSS / JS (jQuery)

A friend of mine who is very passionate about swedish pop music had the idea to create an online christmas calendar which ranked the year's 24 best albums.

The idea came in a bit late which made this hobby project a bit rushed but I am still pretty happy with the result. I was originally not supposed to do the design either. A lot can still be improved and optimized (animations are really slow in some cases), but that's for next christmas.

The design was mocked-up by prototyping with Bootstrap which eventually turned into the final production code. The excellent Isotope library provided the masonry-like layout.

Check it out!
Data Sheet: AMTAB Weighing In Motion (2013)
InDesign, Illustrator, Photoshop

A company I was working for needed a data sheet that our salespeople and reps could show to customers. I was not really prepared for this assignment but in the end I surprised myself with the result. This is the first "design only" work I've done which did not involve some kind of programming. I learned a lot and I would like to do something similar again.

Most images are made from scratch in Illustrator while the overall page design was made in InDesign. It was then translated into Swedish, English and Russian and optimized for print.
